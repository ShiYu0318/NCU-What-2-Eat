## 功能特色

### 餐廳瀏覽
- 後門 / 宵夜街
- 依時間篩選（目前營業中 / 自訂時間）
- 依類型篩選（便當、麵食、飲料、甜點 ......）
- 依價位篩選（$ / $$ / $$$）
- 餐廳詳情：照片集、菜單、Google Maps

### 個人轉盤
- 勾選今天想吃的餐廳
- 圓形轉盤隨機抽取

### 群組模式
- 無需註冊，輸入暱稱即可加入
- 分享連結邀請朋友
- 即時同步每個人的選擇狀態
- 所有人完成選擇後，房主開轉
- 結果同步顯示在所有人的畫面
